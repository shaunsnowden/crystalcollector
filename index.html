CRYSTAL COLLECTOR

Rules and Strategy

Find the diamonds in the rough


The object of the game is to hit the target score each round by clicking each of the crystals. Each round, the crystals will be assigned a random value between 2 and 12. Each crystal can be clicked as many times as you like in order to reach the target score. You have to hit the target "depth" exactly in order to win the round. Crystals are fragile! If you dig too far (i.e. go over the target score), you lose. The values of the crystals will not be shown, so keep track of how much the score accumulates. Good Luck!
